Detailed Description

Routines to deal with nonconvex QPs.

Author
Ben Hermans

The functions in this file serve to set up QPALM for a nonconvex QP. The main routine in this file computes the minimum eigenvalue of a square matrix, based on power iterations. Furthermore, some setting updates are performed.
